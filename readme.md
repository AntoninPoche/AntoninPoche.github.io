# Repository for my researcher website

[Link to the site](https://antoninpoche.github.io/)

## Run locally

```bash
# create env
python -m venv .venv

# activate env
.\.venv\Scripts\activate

# install mkdocs
pip install mkdocs

# build the doc locally
mkdocs serve
```

[Link to the local site](http://127.0.0.1:8000/)
